Docker笔记

 2018-05-05(星期六) 00:00:00  —  sidewalk

基本概念

docker包括三个基本概念 镜像（Image） 容器（Container） * 仓库（Repository）

命令

  • brew cask install docker：mac安装docker
  • docker --version：查看docker版本
  • docker info
  • docker start dmp：启动容器
  • docker stop dmp：停止容器
  • docker restart dmp：重启容器
  • docker rm dmp：删除容器（必须先停止）
  • docker rmi ubuntu：删除镜像
  • docker logs -f dmp：查看容器内部标准输出
  • docker ps：查看当前运行容器
  • docker ps …

用户配置

  • git config --global user.name 'sidewalk'
  • git config --global user.email '491187637@qq.com'
  • 在某个项目中配置不同用户信息则去掉global
  • 配置编辑器：git config core.editor vim

名词

  • HEAD：指向当前版本
  • reflog：命令历史
  • checkout：版本库替换工作区
  • reset：回退版本/把暂存区的修改回退到工作区
  • revert：反向提交

命令

  • 新建并切换分支：

    • git checkout -b feature/abc
    • git checkout -b …
